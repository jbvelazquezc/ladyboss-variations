== Description ==

Plugin to add customizable buttons to variation products.

== Installation ==

1. Make sure you have WooCommerce 3.0.0+ installed
2. Upload `ladyboss-variations.php` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= What Wordpress version is this plugin compatible with ? =

Wordpress 3.0.0+
